Simple tool for adjusting time on Linux. Sometimes stations have their clock messed up and it is difficult
to work them without adjusting the time. Adjtime provides a simple GUI for adjusting the time. Once the QSO
is over, make sure to re-enable the automatic (NTP) time adjustment. Compile with "make" and install under
/usr/local with "make install".

The buttons function as follows:

"Up 0.1 s" step computer's clock forward by 0.1 seconds.
"Down 0.1 s" step computer's clock backward by 0.1 seconds.
"Auto ON/OFF" Turn on/off automatic time synchronization.

Note: When the auto button reads "Auto OFF" it means that Auto is on and pressing the button will turn it off (and vice versa).
